# Pokedex

This is a simple CLI program made for boot.dev's Pokedev's guided project.
<br>It uses the PokeApi to get data about areas and pokemon and print them to the user.
<br>Available commands:
- help [Shows available commands]
- map [Shows the next 20 locations]
- mapb [Shows the previous 20 locations]
- explore [Shows which pokemon can be caught in a certain location]
- catch [Attempts to catch a pokemon and add it to the pokedex]
- inspect [Prints a list of caught pokemon stats]
- pokedex [Prints a list of caught pokemon]
